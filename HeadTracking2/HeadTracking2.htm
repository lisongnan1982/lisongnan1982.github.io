This website is to promote a real-time head pose tracking system, which will be described in the following paper:

[1] Songnan Li, King Ngi Ngan, and Lu Sheng, "Real-time head pose tracking with online face template reconstruction", under review.

Several demo videos are given below which will be included in the final website as well:

Demo 1: (qualitative evaluation)

 

We used a RGB-Depth sensor to track the head pose in 3-dimensional space. Both translational and rotational head movement can be tracked robustly with a tracking speed over 30fps. In the demo video, the green object is a user-specific face template. The face template reconstruction window is shown in the upper right corner occasionally. The system works well when the user wears a glass. In addition, it can handle large occlusion and recover from tracking failure very quickly.

 

Demo 2: (qualitative evaluation)

 

The system can track different users by reconstructing each user a specific face template during the tracking.

 

Demo 3: (application)

 

Using the head pose tracking system, we developed a holographic-like displaying demo. A hand-held camera was used to capture the user's view, as shown in the first part of the video. Notice that the rendered view changes when the user's head moves. The second part of the video is actually captured by the abovementioned hand-held camera, which recorded the subject's view. As the user moves his/her head around, the rendered view changes accordingly showing different sides of the CG object (a dinosaur). The final effect is that the dinosaur looks holographic-like.

 

Demo 4: (application)

 

This application is related to the human-computer interaction technique. We used the head rotations (yaw and pitch) tracked by our head pose tracking system to control the cursor's movement on the 2D screen without using a mouse. In this video, the purple circle simulated the cursor. We also implemented voluntary eye blinking detection to simulate the mouse clicking. The picture will change its color when one of the buttons on the top of the screen is clicked.
